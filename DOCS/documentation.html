HTML Structure

Basic HTML scructure for index.html consists of sections that contain the building div Bootstrap blocks e.g. class="container" class="row" class="span12" etc. Section id's are tied to the background and parallax scrolling e.g. <section id="header-section"> These id's are required for the design of these sections, height, parallax effects, transitions effects of some elements

Here is the HTML code of index.html 
<!--START MAIN-WRAPPER--> 
<div class="main-wrapper">
	<!-- START HEADER headertop -->
	<section class="headertop needhead" id="header-section">
		
		<!-- START TOP MENU -->
         <nav class="nav-top">
               ~ menu content ~
          </nav>
        <!-- END TOP MENU -->
</section>

<!-- START SECTION-->
<section  id="features-section" >
	<div class="container">
		<div class="row-fluid">
			~ Section content  here  ~
		</div>
	</div>
</section>
<!-- END SECTION -->

~ ....... ALL SECTIONS GOES HERE ............~

<!-- FOOTER --> 
    <footer class="footer">
		<section id="contact-section" class="arrow_box">
		~ section content here ~
		</section>
			
		<section id="map-section">
		~ MAP div GOES HERE ~
		</section>
		
    </footer>
    <!-- END FOOTER-->
		 
</div>
<!-- END MAIN-WRAPPER -->

See more info about all Bootstrap plugins here

Some parts of the plugins for it initiation are in the file custom.js, the other goes in the body of html file.

Some plugins require special code to init them. You will find all of activated pluging parts in file ../assets/js/custom.js . E.g., here is the javascript code to initiated scroll and parallax plugin 
							
<script>
jQuery(document).ready(function(){
	jQuery('#nav').localScroll(3000);
	
	//.parallax(xPosition, speedFactor, outerHeight) options:
	//xPosition - Horizontal position of the element
	//inertia - speed to move relative to vertical scroll. Example: 0.1 is one tenth the speed of scrolling, 2 is twice the speed of scrolling
	//outerHeight (true/false) - Whether or not jQuery should use it's outerHeight option to determine when a section is in the viewport
	
	// If you need parallax bg effect just add code like this here ~..  jQuery('#header-section').parallax("50%", 0.1);
					  
</script>
Inview plugin - Event that is fired as soon as an element appears in the user's viewport. 
							
<script>
	jQuery('#show1').bind('inview', function (event, visible) {
        if (visible == true) {
            jQuery(this).addClass("animated fadeInRightBig");

			
        }else{
            jQuery(this).removeClass("animated fadeInRightBig");

			//jQuery('.effect-box').unbind('inview');
        }
    });
</script>
